Actúa como un Diseñador UI/UX y Desarrollador Frontend. Tu tarea es generar y guardar las imágenes de presentación (mockups) de este mismo proyecto para usarlas en nuestro portfolio principal.

Sigue detenidamente estos pasos:
1. Asegúrate de que el servidor de desarrollo esté corriendo u obtén la URL de producción (Vercel).
2. Usa tu herramienta de navegador (Browser Subagent / Puppeteer) para visitar las vistas principales de esta plataforma (Inicio, Dashboard, etc.).
3. Toma capturas de pantalla de alta resolución o pantalla completa, asegurándote de capturar bien la estética "Dark Mode / Deep Space".
4. (Opcional pero recomendado) Envuelve estas capturas en una "carcasa" o ventana de navegador minimalista u oscuro usando CSS/Canvas o un script en Node. 
5. Exporta las imágenes finales optimizadas (en formato `webp` o `png` ligero) en una carpeta llamada `portfolio-exports/` en la raíz de este proyecto.

*Nota:* Si no cuentas con una herramienta para incrustar el marco de navegador/celular, simplemente toma las mejores capturas de pantalla limpias y de tamaño uniforme para que yo pueda pasarlas directamente a nuestra landing principal.
